## [0.1.0] - 2025-01-XX

### Added

- Initial release
- `isAvailable()` - Check if Apple Intelligence is available
- `getAvailabilityStatus()` - Get detailed availability information
- `generateResponse()` - Generate text using on-device LLM
- `createSession()` - Create persistent conversation sessions
- Session management with `sendMessage()` and `reset()`
- Android stub that returns "unsupported" status
- Comprehensive TypeScript types
- Full documentation

### Notes

- Requires iOS 26+ and Apple Silicon devices
- iOS 26 is currently in beta
